# prototypes

The Void - minimalist note-taking/thinking/brainstorming tool
- To run, cd into the_void and type: ```python the_void.py``` (you may have to pip install a few packages)
- Recommended usage = type as you think, and press Enter to cycle through things you've typed.
- Graph structure constructed in the background which you can view (type ? for commands)
- Save, archive and create new sessions
- Other features include search navigation, condensing node + neighbors, and tournament-style node picker
